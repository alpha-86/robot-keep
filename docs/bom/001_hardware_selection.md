## 2. V2.0 硬件方案

### 2.1 主控板对比

#### 选型A: Raspberry Pi 5 8GB ⭐推荐

| 指标 | 参数 |
|------|------|
| 处理器 | Broadcom BCM2712, 4核A76 @ 2.4GHz |
| 内存 | 8GB LPDDR4X |
| GPU | VideoCore VI (支持OpenGL ES) |
| 接口 | 2xUSB 3.0, 2xUSB 2.0, HDMI, CSI, DSI, GPIO |
| 价格 | ¥600（散片）/ ¥700（套装） |
| 功耗 | 5V 5A（25W） |

**优点：**
- 生态最成熟，文档最全
- 8GB内存足够跑所有算法
- CSI接口原生支持Pi Camera
- 可加Edge TPU加速AI

**缺点：**
- 无内置NPU，AI推理依赖CPU

**V2.0匹配度：** ⭐⭐⭐⭐⭐

---

#### 选型B: Raspberry Pi 4B 4GB

| 指标 | 参数 |
|------|------|
| 处理器 | Broadcom BCM2711, 4核A72 @ 1.8GHz |
| 内存 | 4GB LPDDR4 |
| 价格 | ¥350（散片） |
| 功耗 | 5V 3A（15W） |

**优点：**
- 价格便宜
- 生态成熟

**缺点：**
- 处理器较老（A72 vs A76）

**V2.0匹配度：** ⭐⭐⭐⭐（性价比选择）

### 6.1 你在哪个阶段？

| 情况 | 推荐方案 | 理由 |
|------|---------|------|
| 还没验证过算法 | **V1.0** (¥0) | 先用现有设备验证 |
| 已有MiniMax API | **V1.0** (¥0) | 直接开始 |
| 算法已验证，想采购硬件 | **V2.0 极简版** (¥580) | 最低成本验证 |
| 想完整验证产品 | **V2.0 推荐版** (¥1300) | 一步到位 |
